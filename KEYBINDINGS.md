# Keybindings

Leader key: `Space`

## General

| Key | Action |
|-----|--------|
| `<Esc>` | Clear search highlight |
| `<leader>pv` | Open file tree (Oil) |
| `<C-h/j/k/l>` | Move focus between splits |

## Diagnostics

| Key | Action |
|-----|--------|
| `[d` / `]d` | Previous / Next diagnostic |
| `<leader>e` | Show diagnostic float |
| `<leader>q` | Diagnostic quickfix list |

## Finder

| Key | Action |
|-----|--------|
| `<leader>sf` | Search files |
| `<leader>sg` | Search with grep |
| `<leader>sk` | Search keymaps |
| `<leader>sd` | Search diagnostics |
| `fz` | Fuzzy grep |
| `fc` | Search current word |

## LSP

Active when an LSP server is attached to the buffer.

| Key | Action |
|-----|--------|
| `gd` | Go to definition |
| `gD` | Go to declaration |
| `gr` | Go to references |
| `gI` | Go to implementation |
| `K` | Hover documentation |
| `<leader>D` | Type definition |
| `<leader>ds` | Document symbols |
| `<leader>ws` | Workspace symbols |
| `<leader>rn` | Rename |
| `<leader>ca` | Code action |
| `<leader>th` | Toggle inlay hints |

## Completion

Insert mode mappings for the completion menu.

| Key | Action |
|-----|--------|
| `<C-n>` / `<C-p>` | Next / Previous item |
| `<C-y>` | Confirm selection |
| `<C-Space>` | Trigger completion |
| `<C-b>` / `<C-f>` | Scroll docs up / down |
| `<C-l>` / `<C-h>` | Snippet jump forward / back |

## Harpoon

| Key | Action |
|-----|--------|
| `<leader>H` | Add file to harpoon |
| `<leader>h` | Toggle harpoon menu |
| `<leader>1-5` | Jump to harpoon file 1-5 |

## Formatting

| Key | Action |
|-----|--------|
| `<leader>f` | Format buffer |
